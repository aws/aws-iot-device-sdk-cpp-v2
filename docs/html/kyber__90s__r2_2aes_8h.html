AES counter mode keystream generator. A scheduled generated by OQS_AES256_CTR_load_schedule() is passed rather then a key.

Handles a 12- or 16-byte IV. If a 12-byte IV is given, then 4 counter bytes are initialized to all zeros.

Parameters
iv12- or 16-byte initialization vector.
iv_lenLengh of IV in bytes.
scheduleAbstract data structure for a key schedule.
outPointer to a block of memory which is big enough to contain out_len bytes; the result will be written here.
out_lenLength of output bytes to generate.
