AWS_HTTP_API int aws_websocket_decoder_process ( struct aws_websocket_decoder *  decoder,
struct aws_byte_cursor *  data,
bool *  frame_complete 
)

Returns when all data is processed, or a frame and its payload have completed. data will be advanced to reflect the amount of data processed by this call. frame_complete will be set true if this call returned due to completion of a frame. The on_frame and on_payload callbacks may each be invoked once as a result of this call. If an error occurs, the decoder is invalid forevermore.
