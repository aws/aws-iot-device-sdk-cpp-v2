AWS_STATIC_IMPL int aws_mul_u32_checked ( uint32_t  a,
uint32_t  b,
uint32_t *  r 
)

If a * b overflows, returns AWS_OP_ERR; otherwise multiplies a * b, returns the result in *r, and returns AWS_OP_SUCCESS.

Note: We use SETNO which only takes a byte register. To make this easy, we'll write it to dl (which we throw away anyway) and mask off the high bits.
