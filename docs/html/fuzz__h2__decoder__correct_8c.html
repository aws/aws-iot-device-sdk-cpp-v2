AWS_EXTERN_C_BEGIN int LLVMFuzzerTestOneInput ( const uint8_t *  data,
size_t  size

This test generates valid frames from the random input. It feeds these frames through the encoder and ensures that they're output without error. Then it feeds the encoder's output to the decoder and ensures that it does not report an error. It does not currently investigate the outputs to see if they line up with they inputs, it just checks for errors from the encoder & decoder.
