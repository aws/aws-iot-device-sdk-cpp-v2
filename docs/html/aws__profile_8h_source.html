16/*
17 * A set of data types that model the aws profile specification
18 *
19 * A profile collection is a collection of zero or more named profiles
20 * Each profile is a set of properties (named key-value pairs)
21 * Empty-valued properties may have sub properties (named key-value pairs)
22 *
23 * Resolution rules exist to determine what profile to use, what files to
24 * read profile collections from, and what types of credentials have priority.
25 *
26 * The profile specification is informally defined as "what the aws cli does" and
27 * formally defined in internal aws documents.
28 */
