◆ aws_h2_connection_on_stream_closed()

int aws_h2_connection_on_stream_closed ( struct aws_h2_connection *  connection,
struct aws_h2_stream *  stream,
enum aws_h2_stream_closed_when  closed_when,
int  aws_error_code 
)

Invoked immediately after a stream enters the CLOSED state. The connection will remove the stream from its "active" datastructures, guaranteeing that no further decoder callbacks are invoked on the stream.

This should NOT be invoked in the case of a "Connection Error", though a "Stream Error", in which a RST_STREAM is sent and the stream is closed early, would invoke this.

Internal function for bringing connection to a stop. Invoked multiple times, including when:

  • Channel is shutting down in the read direction.
