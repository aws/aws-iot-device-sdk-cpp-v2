#define AWS_CONDITION_VARIABLE_INIT    { .condition_handle = PTHREAD_COND_INITIALIZER, .initialized = true }

Static initializer for condition variable. You can do something like struct aws_condition_variable var = AWS_CONDITION_VARIABLE_INIT;

If on Windows and you get an error about AWS_CONDITION_VARIABLE_INIT being undefined, please include Windows.h to get CONDITION_VARIABLE_INIT.
