Detailed Description

The websocket bootstrap brings a websocket connection into this world, and sees it out again. Spins up an HTTP client, performs the opening handshake (HTTP Upgrade request), creates the websocket handler, and inserts it into the channel. The bootstrap is responsible for firing the on_connection_setup and on_connection_shutdown callbacks.
