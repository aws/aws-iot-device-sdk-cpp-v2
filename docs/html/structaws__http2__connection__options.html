Detailed Description

Options specific to HTTP/2 connections. Initialize with AWS_HTTP2_CONNECTION_OPTIONS_INIT to set default values.

◆ max_closed_streams

size_t aws_http2_connection_options::max_closed_streams

Optional The max number of recently-closed streams to remember. A default number is set by AWS_HTTP2_CONNECTION_OPTIONS_INIT.

If the connection receives a frame for a closed stream, the frame will be ignored or cause a connection error, depending on the frame type and how the stream was closed. Remembering more streams reduces the chances that a late frame causes a connection error, but costs some memory.
