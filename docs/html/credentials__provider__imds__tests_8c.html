◆ IMDS_MAX_REQUESTS

#define IMDS_MAX_REQUESTS   (8)

Copyright Amazon.com, Inc. or its affiliates. All Rights Reserved. SPDX-License-Identifier: Apache-2.0. The max requests SDK could make to IMDS V2 should be 4.

  1. query_role_name -> Get 401, unauthorized, then switch to secure way.
  2. query_token
  3. query role name
  4. query role

By default, the requests made is 3.

  1. query_token (unless gets 400, no matter succeed or not, next step is query role name w/o token)
  2. query role name.
  3. query role

Well, IMDS could act crazy then client would keep switching between secure and insecure way. We will not handle this extreme case.
