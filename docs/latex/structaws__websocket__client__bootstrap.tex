\doxysubsection{Detailed Description}
The websocket bootstrap brings a websocket connection into this world, and sees it out again. Spins up an HTTP client, performs the opening handshake (HTTP Upgrade request), creates the websocket handler, and inserts it into the channel. The bootstrap is responsible for firing the on\+\_\+connection\+\_\+setup and on\+\_\+connection\+\_\+shutdown callbacks.
