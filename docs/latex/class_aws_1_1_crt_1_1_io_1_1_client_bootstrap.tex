\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_aws_1_1_crt_1_1_io_1_1_client_bootstrap}{Client\+Bootstrap}} handles creation and setup of socket connections to specific endpoints.

Note that \mbox{\hyperlink{class_aws_1_1_crt_1_1_io_1_1_client_bootstrap}{Client\+Bootstrap}} may not clean up all its behind-\/the-\/scenes resources immediately upon destruction. If you need to know when behind-\/the-\/scenes shutdown is complete, use \mbox{\hyperlink{class_aws_1_1_crt_1_1_io_1_1_client_bootstrap_ad33fec27c1600aaeb8e30ec32b1302c9}{Set\+Shutdown\+Complete\+Callback()}} or \mbox{\hyperlink{class_aws_1_1_crt_1_1_io_1_1_client_bootstrap_a0021ef57bec8381f73c69e5e4c46c0e1}{Enable\+Blocking\+Shutdown()}} (only safe on main thread).
