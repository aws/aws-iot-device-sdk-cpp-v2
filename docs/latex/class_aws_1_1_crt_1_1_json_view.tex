\doxysubsection{Detailed Description}
Provides read-\/only view to an existing \mbox{\hyperlink{class_aws_1_1_crt_1_1_json_object}{Json\+Object}}. This allows lightweight copying without making deep copies of the \mbox{\hyperlink{class_aws_1_1_crt_1_1_json_object}{Json\+Object}}. Note\+: This class does not extend the lifetime of the given \mbox{\hyperlink{class_aws_1_1_crt_1_1_json_object}{Json\+Object}}. It\textquotesingle{}s your responsibility to ensure the lifetime of the \mbox{\hyperlink{class_aws_1_1_crt_1_1_json_object}{Json\+Object}} is extended beyond the lifetime of its view.
