\doxysubsection{Detailed Description}
Represents a unique configuration for connecting to a single endpoint. You can use a single instance of this class PER endpoint you want to connect to. This object must live through the lifetime of your connection.

Creates a client configuration for use with making new AWS \mbox{\hyperlink{namespace_aws_1_1_iot}{Iot}} specific MQTT Connections with web sockets. interceptor\+: a callback invoked during web socket handshake giving you the opportunity to mutate the request for authorization/signing purposes. If not specified, it\textquotesingle{}s assumed you don\textquotesingle{}t need to sign the request. proxy\+Options\+: optional, if you want to use a proxy with websockets, specify the configuration options here.

If proxy options are used, the tls\+Context is applied to the connection to the remote endpoint, NOT the proxy. To make a tls connection to the proxy itself, you\textquotesingle{}ll want to specify tls options in proxy\+Options.
