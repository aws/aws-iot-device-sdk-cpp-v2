\doxysubsection{Detailed Description}
Streaming \mbox{\hyperlink{class_aws_1_1_crt_1_1_crypto_1_1_hash}{Hash}} object. The typical use case is for computing the hash of an object that is too large to load into memory. You can call \mbox{\hyperlink{class_aws_1_1_crt_1_1_crypto_1_1_hash_a0212c998a976cbef12cb4732dbc574a8}{Update()}} multiple times as you load chunks of data into memory. When you\textquotesingle{}re finished simply call \mbox{\hyperlink{class_aws_1_1_crt_1_1_crypto_1_1_hash_a7979342835e5c7a5171dfee40e4034e6}{Digest()}}. After \mbox{\hyperlink{class_aws_1_1_crt_1_1_crypto_1_1_hash_a7979342835e5c7a5171dfee40e4034e6}{Digest()}} is called, this object is no longer usable.
