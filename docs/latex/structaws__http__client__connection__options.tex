\doxysubsubsection{\texorpdfstring{manual\_window\_management}{manual\_window\_management}}
{\footnotesize\ttfamily bool aws\+\_\+http\+\_\+client\+\_\+connection\+\_\+options\+::manual\+\_\+window\+\_\+management}

Set to true to manually manage the flow-\/control window of each stream.

If false, the connection will maintain its flow-\/control windows such that no back-\/pressure is applied and data arrives as fast as possible.

If true, the flow-\/control window of each stream will shrink as body data is received (headers, padding, and other metadata do not affect the window). {\ttfamily initial\+\_\+window\+\_\+size} determines the starting size of each stream\textquotesingle{}s window. If a stream\textquotesingle{}s flow-\/control window reaches 0, no further data will be received. The user must call \mbox{\hyperlink{request__response_8h_a6b4f85fec6a60770fa25e204f6db2886}{aws\+\_\+http\+\_\+stream\+\_\+update\+\_\+window()}} to increment the stream\textquotesingle{}s window and keep data flowing. \mbox{\Hypertarget{structaws__http__client__connection__options_afd7897f9d2decd75527c0d5f8cbdc36e}\label{structaws__http__client__connection__options_afd7897f9d2decd75527c0d5f8cbdc36e}}
