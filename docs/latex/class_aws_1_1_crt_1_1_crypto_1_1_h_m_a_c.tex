\doxysubsection{Detailed Description}
Streaming \mbox{\hyperlink{class_aws_1_1_crt_1_1_crypto_1_1_h_m_a_c}{HMAC}} object. The typical use case is for computing the \mbox{\hyperlink{class_aws_1_1_crt_1_1_crypto_1_1_h_m_a_c}{HMAC}} of an object that is too large to load into memory. You can call \mbox{\hyperlink{class_aws_1_1_crt_1_1_crypto_1_1_h_m_a_c_a08824f48908333900e129b7a27f37646}{Update()}} multiple times as you load chunks of data into memory. When you\textquotesingle{}re finished simply call \mbox{\hyperlink{class_aws_1_1_crt_1_1_crypto_1_1_h_m_a_c_a9b9d7644f2e886b33fdd9be59d3bbacf}{Digest()}}. After \mbox{\hyperlink{class_aws_1_1_crt_1_1_crypto_1_1_h_m_a_c_a9b9d7644f2e886b33fdd9be59d3bbacf}{Digest()}} is called, this object is no longer usable.
