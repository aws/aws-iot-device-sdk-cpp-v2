\doxysubsection{Detailed Description}
Represents a persistent \mbox{\hyperlink{namespace_aws_1_1_crt_1_1_mqtt}{Mqtt}} Connection. The memory is owned by \mbox{\hyperlink{class_aws_1_1_crt_1_1_mqtt_1_1_mqtt_client}{Mqtt\+Client}}. Unless specified all function arguments need only to live through the duration of the function call.
