This function overrides the original implementation of the memcpy function from \mbox{\hyperlink{string_8h}{string.\+h}}. It copies the values of n bytes from the $\ast$src to the $\ast$dst. It also checks if the size of the arrays pointed to by both the $\ast$dst and $\ast$src parameters are at least n bytes and if they overlap. Override the version of memcpy used by CBMC.
