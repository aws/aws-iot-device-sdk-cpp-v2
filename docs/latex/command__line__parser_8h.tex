A mostly compliant implementation of posix getopt\+\_\+long(). Parses command-\/line arguments. argc is the number of command line arguments passed in argv. optstring contains the legitimate option characters. The option characters coorespond to \mbox{\hyperlink{structaws__cli__option_a1580617637a8ab9499f714841f29ffff}{aws\+\_\+cli\+\_\+option\+::val}}. If the character is followed by a \+:, the option requires an argument. If it is followed by \textquotesingle{}\+::\textquotesingle{}, the argument is optional (not implemented yet).

longopts, is an array of struct \mbox{\hyperlink{structaws__cli__option}{aws\+\_\+cli\+\_\+option}}. These are the allowed options for the program. The last member of the array must be zero initialized.

If longindex is non-\/null, it will be set to the index in longopts, for the found option.

Returns option val if it was found, \textquotesingle{}?\textquotesingle{} if an option was encountered that was not specified in the option string, returns -\/1 when all arguments that can be parsed have been parsed.
