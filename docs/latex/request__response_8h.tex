Increment the stream\textquotesingle{}s flow-\/control window to keep data flowing.

If the connection was created with {\ttfamily manual\+\_\+window\+\_\+management} set true, the flow-\/control window of each stream will shrink as body data is received (headers, padding, and other metadata do not affect the window). The connection\textquotesingle{}s {\ttfamily initial\+\_\+window\+\_\+size} determines the starting size of each stream\textquotesingle{}s window. If a stream\textquotesingle{}s flow-\/control window reaches 0, no further data will be received.

If {\ttfamily manual\+\_\+window\+\_\+management} is false, this call will have no effect. The connection maintains its flow-\/control windows such that no back-\/pressure is applied and data arrives as fast as possible.
