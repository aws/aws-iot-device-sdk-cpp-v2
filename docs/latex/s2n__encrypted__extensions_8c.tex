In all handshakes, the server MUST send the Encrypted\+Extensions message immediately after the Server\+Hello message.

The Encrypted\+Extensions message contains extensions that can be protected, i.\+e., any which are not needed to establish the cryptographic context but which are not associated with individual certificates.
