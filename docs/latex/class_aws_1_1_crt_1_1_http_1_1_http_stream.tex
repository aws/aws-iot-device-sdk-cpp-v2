{\footnotesize\ttfamily void Aws\+::\+Crt\+::\+Http\+::\+Http\+Stream\+::\+Update\+Window (\begin{DoxyParamCaption}\item[{std\+::size\+\_\+t}]{increment\+Size }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [noexcept]}}

Updates the read window on the connection. In \mbox{\hyperlink{namespace_aws_1_1_crt_1_1_http}{Http}} 1.\+1 this relieves TCP back pressure, in H2 this will trigger two WINDOW\+\_\+\+UPDATE frames, one for the connection and one for the stream.

You do not need to call this unless you utilized the {\ttfamily out\+Window\+Update\+Size} in {\ttfamily On\+Incoming\+Body}. See {\ttfamily On\+Incoming\+Body} for more information.

{\ttfamily increment\+Size} is the amount to update the read window by.
