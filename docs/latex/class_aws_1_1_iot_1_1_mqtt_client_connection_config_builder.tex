\doxysubsection{Detailed Description}
Represents configuration parameters for building a \mbox{\hyperlink{class_aws_1_1_iot_1_1_mqtt_client_connection_config}{Mqtt\+Client\+Connection\+Config}} object. You can use a single instance of this class PER \mbox{\hyperlink{class_aws_1_1_iot_1_1_mqtt_client_connection_config}{Mqtt\+Client\+Connection\+Config}} you want to generate. If you want to generate a config for a different endpoint or port etc... you need a new instance of this class.
