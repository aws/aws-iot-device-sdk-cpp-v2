This function overrides the original implementation of the memcpy function from \mbox{\hyperlink{string_8h}{string.\+h}}. It copies the values of n bytes from the $\ast$src to the $\ast$dst. It also checks if the size of the arrays pointed to by both the $\ast$dst and $\ast$src parameters are at least n bytes and if they overlap.

This takes advantage of the fact that 64bit operations require fewer array updates, which can make this version faster than the naive unrolling when used in CBMC. Benchmark your particular proof to know for sure.
