 This document provides information about the AWS IoT device SDK for C++ V2.

If you have any issues or feature requests, please file an issue or pull request.

This SDK is built on the AWS Common Runtime, a collection of libraries (\href{https://github.com/awslabs/aws-c-common}{\texttt{ aws-\/c-\/common}}, \href{https://github.com/awslabs/aws-c-io}{\texttt{ aws-\/c-\/io}}, \href{https://github.com/awslabs/aws-c-mqtt}{\texttt{ aws-\/c-\/mqtt}}, \href{https://github.com/awslabs/aws-c-http}{\texttt{ aws-\/c-\/http}}, \href{https://github.com/awslabs/aws-c-cal}{\texttt{ aws-\/c-\/cal}}, \href{https://github.com/awslabs/aws-c-auth}{\texttt{ aws-\/c-\/auth}}, \href{https://github.com/awslabs/s2n}{\texttt{ s2n}}...) written in C to be cross-\/platform, high-\/performance, secure, and reliable. The libraries are bound to C++ by the \href{https://github.com/awslabs/aws-crt-cpp}{\texttt{ aws-\/crt-\/cpp}} package.
