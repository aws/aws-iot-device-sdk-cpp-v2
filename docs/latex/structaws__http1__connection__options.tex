{\footnotesize\ttfamily size\+\_\+t aws\+\_\+http1\+\_\+connection\+\_\+options\+::read\+\_\+buffer\+\_\+capacity}

Optional Capacity in bytes of the HTTP/1 connection\textquotesingle{}s read buffer. The buffer grows if the flow-\/control window of the incoming HTTP-\/stream reaches zero. If the buffer reaches capacity, no further socket data is read until the HTTP-\/stream\textquotesingle{}s window opens again, allowing data to resume flowing.

Ignored if {\ttfamily manual\+\_\+window\+\_\+management} is false. If zero is specified (the default) then a default capacity is chosen. A capacity that is too small may hinder throughput. A capacity that is too big may waste memory without helping throughput.
